# Robotics Speech

## Requirements
* [Anaconda](https://www.anaconda.com/)
* [SpeechRecognition](https://github.com/Uberi/speech_recognition)

## Instalation

```bash
conda create -n robotics-speech python=3.7.3
conda activate robotics-speech
pip install -r requirements.txt
```

## Running

```bash
conda activate robotics-speech
python src/main.py
```
